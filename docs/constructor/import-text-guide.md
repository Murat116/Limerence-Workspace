# Руководство по импорту текста

## Обзор

Система импорта текста позволяет создавать интерактивные истории из структурированного текста. Поддерживается автоматическое создание сцен, диалогов, вариантов выбора и связей между ними.

## Формат текста

### Основная структура

```
#### Глава 1: Название главы
### Сцена: Название сцены

## Список диалогов 1

#Речь, Имя персонажа, текст реплики
- Вариант 1 (список 2)
- Вариант 2 (список 3)

## Список диалогов 2

#Речь, Другой персонаж, ответ на выбор 1
```

### Элементы структуры

#### Главы
- **Формат**: `#### Глава N: Название`
- **Пример**: `#### Глава 1: Начало истории`
- **Функция**: Создает новую главу в проекте

#### Сцены
- **Формат**: `### Название сцены`
- **Пример**: `### Сцена: Таверна`
- **Функция**: Создает новую сцену внутри текущей главы

#### Списки диалогов
- **Формат**: `## Название списка`
- **Пример**: `## Список диалогов 1`
- **Функция**: Группирует связанные диалоги

#### Диалоги
- **Формат**: `#Тип, Имя персонажа, текст`
- **Пример**: `#Речь, Тихон, Привет! Как дела?`
- **Функция**: Создает реплику персонажа

## Типы диалогов

### Основные типы
- `#Речь` - обычная речь
- `#Мысль` - внутренние мысли
- `#Действие` - описание действий

### Специальные типы
- `#Речь, (читает вслух)` - чтение вслух
- `#Речь, (мысленно)` - мысленная речь
- `#Речь, (внутренний голос)` - внутренний голос
- `#Речь, (шёпотом)` - шепот
- `#Речь, РАДИО` - радиосообщение
- `#Речь, МИЛА (оставляет голосовое сообщение)` - голосовое сообщение

## Варианты выбора

### Формат
```
#Выбор
- Текст варианта 1 (список 2)
- Текст варианта 2 (список 3)
- Текст варианта 3 (список 4)
```

### Связи между вариантами
- **Формат**: `(список N)` в конце варианта
- **Пример**: `- Подойти к незнакомцу (список 2)`
- **Функция**: Указывает, к какому списку диалогов перейти после выбора

## Система связей

### Явные связи

#### 1. Связи через варианты
```
#Выбор
- Вариант 1 (список 2)
- Вариант 2 (список 3)
```
**Результат**: После выбора варианта 1 → переход к списку 2, после варианта 2 → к списку 3

#### 2. Связи через реплики
```
#Речь, Персонаж, Я пойду налево (список 2)
```
**Результат**: После этой реплики → переход к списку 2

#### 3. Связи через заголовки списков
```
## Список диалогов 1 (список 2)
```
**Результат**: После завершения списка 1 → переход к списку 2

### Автоматические связи

#### Последовательные связи
- Если не указаны явные связи, диалоги идут последовательно
- Варианты без указания списка ведут к следующему списку по порядку

#### Скрытие маркеров
- Маркеры `(список N)` автоматически скрываются в интерфейсе
- Пользователь видит только текст без технических маркеров

## Дополнительные возможности

### Уведомления в вариантах
```
#Выбор
- [уведомление: Получено достижение] Выбрать путь воина
- [уведомление: -10 HP] Рискнуть и прыгнуть
```

### Многострочные реплики
```
#Речь, Персонаж, Это первая строка
Это вторая строка
Это третья строка
#Следующий диалог
```

### Имена персонажей в тексте
```
#Речь, Тихон, Привет! Как дела?
#Речь, Анна - Отлично, спасибо!
```
**Результат**: Имя персонажа автоматически обновляется на "Анна"

### Звуковые эффекты
```
#Действие, *звук шагов*
#Действие, *дверь скрипит*
```

## Пример полной истории

```
#### Глава 1: Начало
### Сцена: Таверна

## Список диалогов 1

#Речь, Тихон, Добро пожаловать в нашу таверну!
#Речь, Тихон, Что будете заказывать?

#Выбор
- [уведомление: +5 золота] Заказать эль (список 2)
- [уведомление: +10 золота] Заказать вино (список 3)
- Уйти (список 4)

## Список диалогов 2

#Речь, Тихон, Отличный выбор! Эль у нас лучший в городе.
#Речь, Тихон, Вот ваш заказ, наслаждайтесь!

## Список диалогов 3

#Решь, Тихон, Вино? У нас есть отличное красное вино из южных земель.
#Речь, Тихон, Подаю ваш заказ!

## Список диалогов 4

#Речь, Тихон, Жаль, что вы уходите. Возвращайтесь еще!
```

## Расположение элементов

### Сцены
- Располагаются **горизонтально** с отступом 800 пикселей
- Каждая сцена содержит свои списки диалогов

### Списки диалогов
- Располагаются в **сетке** внутри сцены
- Максимум 3 колонки в ряду
- Отступ между элементами: 400 пикселей

### Диалоги
- Располагаются **вертикально** внутри списка диалогов
- Отступ между диалогами: 200 пикселей

## Обработка ошибок

### Автоматические исправления
- Пустые списки диалогов удаляются
- Персонажи без имени получают имя "Неизвестный"
- Текст без структуры преобразуется в диалоги "Неизвестного"

### Валидация
- Проверка корректности формата
- Предупреждения о возможных проблемах
- Автоматическое исправление мелких ошибок

## Экспорт и редактирование

### Просмотр текущего текста
- Кнопка "Показать текущий текст проекта" в модальном окне импорта
- Отображает весь проект в текстовом формате

### Редактирование
- Кнопка "Редактировать" для изменения текста
- Сохранение изменений с автоматическим обновлением проекта
- Валидация перед сохранением

## Советы по использованию

### Планирование связей
1. Определите основные точки выбора в истории
2. Пронумеруйте списки диалогов последовательно
3. Используйте явные связи `(список N)` для сложных переходов

### Структурирование
1. Начинайте с глав и сцен
2. Группируйте связанные диалоги в списки
3. Используйте варианты для интерактивности

### Тестирование
1. Импортируйте текст и проверьте связи
2. Убедитесь, что все варианты ведут куда нужно
3. Проверьте, что маркеры скрыты в интерфейсе

## Ограничения

### Размер
- Максимальная длина текста: ограничена производительностью браузера
- Рекомендуется разбивать большие истории на части

### Сложность
- Слишком много связей может замедлить отображение
- Рекомендуется не более 50 списков диалогов на сцену

### Совместимость
- Формат оптимизирован для русского языка
- Поддержка других языков ограничена
