[[Исход]] (Variant)

Один из возможных вариантов выбора в [[Диалог]]. Может влиять на [[Статы пользователя]] и ход сюжета. Представляет собой действие пользователя, совершаемое в рамках реплики.

Наследует [[BaseModel]].

⸻

🧩 Поля
•	uuid: См. [[BaseModel]]
•	text: Текст варианта выбора, отображаемый пользователю
•	statupUUID?: UUID связанного [[Улучшение статов]] (если выбор влияет на характеристики)
•	statsUpValue?: Числовое значение влияния на характеристику (если statupUUID указан)

⸻

📚 Поведение
•	Показывается пользователю при наличии variants у [[Диалог]]
•	После выбора:
•	Применяется улучшение статов (если задано)
•	Сохраняется uuid выбранного [[Исход]] для использования в логике:
•	Подбор следующего [[Диалог]]
•	Активация следующих [[Сцена]]
•	Может отображаться как кнопка/опция с дополнительными визуальными элементами (иконка, +1 и т.д.)

⸻

🧠 Пример

💬 “Что ты сделаешь?”

🟢 Защищу товарища
→ [[Улучшение статов]]: Сила +1

🔴 Отступлю
→ Без изменений
