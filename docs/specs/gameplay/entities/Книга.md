[[Книга]] (Story)

Главная сущность, объединяющая всю сюжетную структуру. Содержит последовательные [[Глава]], обложку, описание и информацию об авторе.

Наследует [[BaseModel]].

⸻

Состоит из [[Глава]]

⸻

🧩 Статические поля
•	uuid: См. [[BaseModel]]
•	title: Название книги
•	description: Краткое описание истории
•	imageUrl: Путь к обложке
•	chapters: Список [[Глава]]
•	author: Имя автора
•	styleReference: (author-only, конструктор) Одна строка — общий визуальный стиль истории для генерации ассетов (Style Bible). Подмешивается в промты персонажей и сцен. @see docs/web/scenario-import/style-bible.md

⸻

🧮 Должны быть высчитываемые поля
•	Прогресс прохождения книги — доля завершённых [[Глава]] от общего числа
•	Общие [[Статы пользователя]] — суммарное состояние после всех пройденных глав
•	Список глав — с учётом их текущего status (read, available, locked)

⸻

🗄️ Взаимодействие с БД
•	✅ Сохранение полученных извне данных (например, при загрузке книг с сервера)
•	✅ Получение модели книги по uuid
•	✅ Обновление факта прочтения [[Глава]]
•	Пользователь может:
•	Сбросить прогресс до определённой главы (все главы после — locked)
•	Полностью сбросить прогресс (все главы — locked, кроме первой — available)

⸻

📚 Поведение
1.	Пользователь выбирает книгу из списка
2.	Загружается информация (title, description, imageUrl, author)
3.	Показывается текущий прогресс и статус глав
4.	При старте — начинается первая available [[Глава]]
5.	Прогресс и [[Статы пользователя]] обновляются по мере прохождения
