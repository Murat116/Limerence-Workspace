[[АудиоТрек]] (AudioTrack)

Модель, представляющая аудио-файл для воспроизведения в [[Сцена]] или через [[Диалог]] (DialogAudioEvent).

Наследует [[BaseModel]].

⸻

🧩 Поля
•	uuid: См. [[BaseModel]]
•	url: Путь к локальному аудиофайлу (после download из ProjectAsset или прямой URL).
•	assetId (опционально): UUID ассета из ProjectAsset.
•	isLooped: Зацикленное (`playback_type === 0`) или one-shot воспроизведение.
•	volume: Громкость от 0.0 до 1.0.

⸻

📚 Поведение сцены
•	На сцене используется только **audioTracks[0]** как BGM (`mode: scene_bgm`).
•	При входе в сцену без BGM — гасится только BGM-канал; overlay (SFX) сохраняется.
•	При смене сцены старый BGM заменяется новым.

⸻

📚 DialogAudioEvent (на реплике)
•	`stop_other_play` — stopAll, затем loop ? новый BGM : one-shot overlay.
•	`play_overlay` — SFX/музыка поверх текущего BGM.
•	`stop_all` — полная остановка всех каналов.

⸻

📚 ProjectAsset
•	URL музыки/SFX резолвится через `projectAssets[assetId].url`.
•	Офлайн: локальный path после download (`projectAsset:${assetId}`).

⸻

📚 Cleanup
•	Выход из истории / resetQuest — `AudioService.stopAll()`.
