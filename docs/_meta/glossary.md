# Glossary

| Term | Internal | Meaning |
|------|----------|---------|
| Story Pass | `subscription` | Platform subscription, all premium content |
| Полный доступ | `full_story` | One-time full story purchase |
| Chapter Pass | `chapter_pass` | One chapter: hidden paths + early access + offline |
| Premium variant | `is_premium` | Dialog choice requiring entitlement |
| Early access | `early_access` | Chapter before public `release_date` |
| Entitlement priority | — | `full_story` > `chapter_pass` > `subscription` > free |
| Paywall ID | `PW-01`… | Server-driven paywall scenario |
| `requires_pass_to_read` | — | Blocks reading without pass (constructor setting) |
