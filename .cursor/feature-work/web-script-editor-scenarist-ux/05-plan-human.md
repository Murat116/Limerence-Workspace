## Резюме

Делаем текстовый редактор **удобным для сценариста**: цветовая разметка DSL как в IDE, подсказки при наборе, рассказчик без имени в тексте, inspector-only поля — **чипы** вместо сырого `;`, настройки «воздуха» и шрифта — в Story Settings. Всё в **одном PR** вместе с двумя багами v1 (#8 jump при несохранённом черновике, #9 смена `(список N)`).

**Главные риски:** presentation layer (пустые строки после Apply) — самый хрупкий кусок; meta-widgets не должны ломать parse stat gates.

**planStatus:** draft

**Спека:** `03-approved-spec.md`, `docs/web/text-editor-mode.md#scenarist-ux`

## User flow

### Первый вход в «Текст»

1. Автор открывает историю → режим **«Текст»** (как в v1).
2. Видит главу с **цветными заголовками** сцен/списков, разными акцентами `#Речь` / `#Мысль` / `#Выбор`, приглушёнными вариантами `- …`.
3. Условия веток (`; stat:`, `; variant:`) — **обычный текст** с жёлто-оранжевой подсветкой, не чипы.
4. Между блоками — пустые строки по prefs истории (по умолчанию 1 после сцены и списка).

### Набор реплики

5. Печатает `#` → выпадают типы: `#Речь`, `#Мысль`, `#Выбор`.
6. После `#Речь,` → список персонажей (**без** «Рассказчик»).
7. Рассказчик: `#Речь, Ночь была тихой.` — одна запятая, без имени.
8. Персонаж: `#Речь, Тихон, Привет!` — две запятые.
9. Рядом с репликой — **чипы** (read-only): рамка, позиция, размер шрифта, жирность, курсив. Клик → Inspector.

### Навигация

10. Клик сцены/списка в **outline слева** → прокрутка к строке в редакторе.
11. Курсор в реплике → в outline **подсвечивается родительский список** (и прокручивается к нему).
12. Над редактором **breadcrumb**: `Сцена: Таверна › Список 2 › #Речь`.
13. Слева от номеров строк — **fold** для `###` и `##`.

### Apply

14. Жмёт **Apply** → модалка **preview diff**: «+2 списка, ~3 реплики, −1 сцена», warnings.
15. Confirm → merge в store; **ручные пустые строки** в черновике сохраняются; prefs-отступы re-inject.
16. Если убрал `(список 3)` у варианта — на графе ветка **исчезает** (#9).

### Анализ и jump

17. Жмёт `R` → ошибка → **Перейти к строке**.
18. Если черновик **не сохранён** (dirty) — toast «Сначала Apply» (#8), курсор не уезжает на старую строку.

### Настройки редактора

19. В toolbar: слайдер размера шрифта редактора + ссылка **«Настройки редактора»**.
20. Story Settings → вкладка **«Редактор»**: отступы после сцены/списка/реплики, размер шрифта по умолчанию.
21. Сохранение → `story.script_editor_prefs` в Supabase; применяется при следующем export/load.

## Что увидит пользователь

| Место | Содержимое |
|-------|------------|
| Редактор | IDE-цвета по типу блока; monospace, размер из prefs/slider |
| Stat gates | Редактируемый текст `; stat:` / `; variant:` с warning-подсветкой |
| Meta | Чипы: «Фон: tavern.jpg», «Рамка: default», «16px · semibold · italic» |
| Медиа в чипах | Только имя файла, не полный URL |
| Breadcrumb | Контекст сцена › список › тип реплики |
| Fold gutter | Сворачивание сцены/списка |
| Apply | Preview diff modal перед merge |
| Toolbar | Font slider, «Настройки редактора», Apply/Сброс/Файл |
| Settings «Редактор» | 3 числа blank lines + font size |
| Рассказчик в тексте | `#Речь, текст` без слова «Рассказчик» |
| Outline | Только сцены и списки (реплик нет) |

## Edge cases

| Ситуация | Поведение |
|----------|-----------|
| **Dirty + jump из R** | `apply_first` — не скроллить по устаревшему source map (#8) |
| **Убрал `(список N)`** | Apply сбрасывает `nextDialogId` / `nextAvailableDialogListUUID` (#9) |
| **Старый текст `#Речь, Рассказчик, …`** | Apply нормализует в `#Речь, текст`; в БД `person_uuid = null` |
| **Реплика без рамки** | Чип fontSize не показываем |
| **Inspector меняет фон** | Чип обновляется через overlay; сюжетный текст не трогаем |
| **Ручные пустые строки + Apply** | Сохраняются в presentation layer черновика, не в entity store |
| **Prefs изменили в Settings** | Следующий load/export re-inject отступы; dirty draft — confirm при уходе |
| **Delete с связями** | Preview diff → затем существующий confirm delete |
| **Autocomplete без персонажей** | Только типы реплики и gates |
| **Fold свёрнут + jump** | Развернуть блок цели перед scroll |
| **Офлайн / save story** | Prefs в payload story при save; миграция уже в sync |

## Что НЕ делаем

- Stat gates как нередактируемые теги/chips
- Реплики в outline-дереве
- Plain narration без `#Речь`
- Fountain, live preview, dictation (H1–H6 из brief)
- Отдельный PR только для parent blockers
- `char-narrator` в `person_uuid` в БД
- UUID в autocomplete

## Глоссарий

| Термин | По-русски |
|--------|-----------|
| Presentation layer | «Воздух» в документе — пустые строки поверх сюжета, не в store |
| Meta widget / чип | Read-only метка inspector-only поля в тексте |
| Gate highlight | Подсветка условий `; stat:` / `; variant:` |
| Preview diff | Сводка изменений перед Apply |
| Рассказчик | Реплика без персонажа; в тексте без имени |
| `script_editor_prefs` | Настройки редактора на уровне истории |
| Dirty черновик | Текст изменён, Apply ещё не нажат |
